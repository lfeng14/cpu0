# cpu0 Project Guide

This repository documents the journey of building a backend for the Cpu0 architecture using LLVM. Each chapter covers a specific topic:

- [CH1-Cpu0 Architecture and LLVM Structure](CH1-Cpu0%20Architecture%20and%20LLVM%20Structure.md): Overview of the Cpu0 architecture and how LLVM fits into the design.
- [CH2-Backend structure](CH2-Backend%20structure.md): The overall structure of the backend implementation.
- [CH3-Arithmetic and Logic Instructions](CH3-Arithmetic%20and%20Logic%20Instructions.md): Implementing arithmetic and logical operations.
- [CH4-Generating object files](CH4-Generating%20object%20files.md): Steps to produce object files from the compiler.
- [CH5-Global Variables](CH5-Global%20Variables.md): Handling global variables in the compilation process.
- [CH6-Other data type](CH6-Other%20data%20type.md): Support for additional data types.
- [CH7-Control flow statements](CH7-Control%20flow%20statements.md): Implementing control flow like loops and branches.
- [CH8-Function call](CH8-Function%20call.md): How function calls are managed.

Use this README as a guide to navigate the chapters and understand where to find specific information about each stage of the Cpu0 backend development.
